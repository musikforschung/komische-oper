 # Topography
Topography of concert life in Berlin, 1880-1945

The topography of concert life in Berlin is an interactive map of places where concert life in Berlin was between 1880 and 1945. The application was created on the basis of the Java Script library [Leaflet](https://github.com/Leaflet).
On the map, historical places such as places of residence, training and performance venues as well as locations of the music industry are linked with other information freely available on the world wide web. The identifiers of the [GND](https://www.dnb.de/DE/Professionell/Standardisierung/GND/gnd_node.html) serve as a basis. The World Geodetic System 1984 (WGS 84) was used for the coordinates. Further information and a view of the map can be found [here](https://www.simpk.de/forschung/themen/interpretationsforschung/archiv-des-konzertlebens/topographie-des-berliner-konzertlebens-1880-1945/).

# Which files are included?

* The folders [build](https://github.com/musikforschung/Topographie/tree/main/build), [dist](https://github.com/musikforschung/Topographie/tree/main/dist), [icon](https://github.com/musikforschung/Topographie/tree/main/icon), [lib](https://github.com/musikforschung/Topographie/tree/main/lib), [src](https://github.com/musikforschung/Topographie/tree/main/src) contain the code and additional plugins for the interactive map.
* In the folder [images](https://github.com/musikforschung/Topographie/tree/main/images) photos or videos can be stored, which should be linked in the map with the respective geographic location.
* The files [berlinkarte.js](https://github.com/musikforschung/Topographie/blob/main/berlinkarte.js) and [berlinkarte-mobil.js](https://github.com/musikforschung/Topographie/blob/main/berlinkarte-mobil.js) contain the actual data that are used to display markers, descriptions, images and links on the map are required.
* The files [berlinkarte.html](https://github.com/musikforschung/Topographie/blob/main/berlinkarte.html) and [berlinkarte-mobil.html](https://github.com/musikforschung/Topographie/blob/main/berlinkarte-mobil.js) are required to view the map in an Internet browser.
* In the file [Topographie_Datenblatt.xlsx](https://github.com/musikforschung/Topographie/blob/main/Topographie_Datenblatt.xlsx) all information collected about the places contained in the map is documented. It forms the basis for creating the files [berlinkarte.js](https://github.com/musikforschung/Topographie/blob/main/berlinkarte.js) and [berlinkarte-mobil.js](https://github.com/musikforschung/Topographie/blob/main/berlinkarte-mobil.js).

# How can I use the files?

1. Download all files to a local folder. Keep the structure of the files shown in this repository. The file [Topographie_Datenblatt.xlsx](https://github.com/musikforschung/Topographie/blob/main/Topographie_Datenblatt.xlsx) is not necessary for this.
2. To display your own data on the map, edit the files [berlinkarte.js](https://github.com/musikforschung/Topographie/blob/main/berlinkarte.js) and [berlinkarte-mobil.js](https://github.com/musikforschung/Topographie/blob/main/berlinkarte-mobil.js) or create new ones as required.
